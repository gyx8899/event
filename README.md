# npm template
NPM package project template

## Publish steps

- Login: add user with your `Username`, `Password`, `Email`

```shell script
# First time publish
npm adduser
# Other times publish
npm login
```

- Publish in your git gui or click publish.sh bash file
```shell script
npm publish
```

## Install
```cmd
npm install @daybyday/xxx --save
```

## Importing
```jsx
import xxx from '@daybyday/xxx';
```
